# Git Status Indicator

PhpStorm/IntelliJ plugin that shows a colored icon in the status bar indicating Git repository status:

- **Green** - No uncommitted changes
- **Red** - Uncommitted changes present

Click the icon to open the Commit window.

## Installation

Available on [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/XXXXX-git-status-indicator)

Or install manually: Settings → Plugins → ⚙️ → Install Plugin from Disk
